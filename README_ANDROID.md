# 📱 دليل بناء وتشغيل تطبيق القرآن الكريم كامل للأندرويد

مرحباً بك! لقد تم تحويل مشروع القرآن الكريم والخدمات الإسلامية بالكامل إلى **تطبيق أندرويد احترافي متكامل** ومبني باستخدام لغة **Kotlin** ودعم لـ **Material Design 3** بالتعاون مع أحدث تقنيات الويب المحلية السريعة (Offline-First WebView).

---

## ✨ مميزات نسخة الأندرويد الاحترافية

1. **دعم كامل لـ Material Design 3:** يعتمد قالب التطبيق الأساسي على أحدث سمات ماتيريال ديزاين لتوفير حواف دائرية أنيقة وحركات انسيابية.
2. **شاشة ترحيبية أصلية (Native Splash Screen):** شاشة ترحيبية سريعة بلمسات إسلامية باللون الأخضر والذهبي، تفتح لمدة 2.5 ثانية مع مؤشر دوران متحرك قبل الإقلاع.
3. **أداء Offline-First فائق السرعة:** يتم تجميع كافة ملفات الويب وحشوها داخل مجلد الأصول الجغرافي للأندرويد (`assets`)، مما يعني أن واجهات التطبيق، السور الكريمة، والأذكار تعمل **كاملةً بدون إنترنت وبسرعة مذهلة.**
4. **شريط تحميل علوي تفاعلي (Progress Bar):** عند تجولك بالتطبيق، يعطيك شريط رقيق مذهب أعلى الشاشة انطباعاً بمقدار تقدم تحميل بيانات الصوت والأثير بدقة.
5. **مزامنة صلاحية الموقع الجغرافي (Geolocation API):** يدمج التطبيق كود برمجيات `WebChromeClient` خاص بأندرويد، يعترض طلبات الموقع لربطها وتفعيلها تلقائياً لدعم **اتجاه القبلة ومواقيت الصلاة الدقيقة** بحسب موقع المصلي الحالي.
6. **دعم تحديث الصفحة بالسحب (Swipe to Refresh):** محاط بواجهة سحب تفاعلية تتيح لك إعادة تحميل الصفحة في حال حدوث أي انقطاع صوتي أو طلب تنشيط البيانات بسلاسة فائقة.
7. **الرجوع الذكي (Smart Back Button):** يعترض التطبيق زر الرجوع الفيزيائي للهاتف؛ فإذا كنت داخل صفحات فرعية للقرآن أو الأذكار يرجع بك خطوة للخلف داخل التطبيق بدلاً من إغلاق التطبيق فجأة.
8. **أيقونة احترافية كاملة الدعم (Adaptive Icons):** تم دمج أيقونتك الفنية المذهلة المصممة للقرآن والقبلة لتبدو ممتازة على شاشات كافة هواتف الأندرويد (الحديثة والقديمة).

## 📂 هيكلية مشروع الأندرويد

تتسق ملفات المشروع مباشرة داخل مجلد `android/` بالهيكلية الرسمية لـ **Android Studio**:

* **`android/build.gradle` & `settings.gradle`:** ملفات إعداد Gradle الأساسية مع تفعيل مكونات Kotlin الروتينية.
* **`android/app/build.gradle`:** تهيئة إصدارات الـ SDK (يدعم من **Android 8.0 Oreo - SDK 26** وحتى أحدث إصدار **Android 14 - SDK 34**).
* **`android/app/src/main/AndroidManifest.xml`:** يطلب الصلاحيات المطلوبة: الإنترنت، الموقع الدقيق والتقريبي، الكاميرا والوصول للإشعارات على Android 13+.
* **`android/app/src/main/java/com/example/app/`:**
  * **`SplashActivity.kt`:** شاشة الترحيب الأصلية التي تمهد الدخول الفخم للتطبيق.
  * **`MainActivity.kt`:** محرك الواجهات والـ WebView المعالج بكفاءة وأمان.
* **`android/app/src/main/res/`:** الموارد المرئية (Layouts, Strings, Colors, XML Themes).
* **`android/app/src/main/assets/dist/`:** واجهات الويب المصدرية المجمّعة بالكامل لتشغيل التطبيق بدون خوادم خارجية وبأعلى ثبات.

## 🚀 طريقة التجميع والبناء خطوة بخطوة

### 1️⃣ مزامنة أصول الويب مع تطبيق الأندرويد
أول خطوة هي بناء كود واجهات الويب ونسخها تلقائياً لمجلد الأصول بالأندرويد عبر تشغيل ملف الأتمتة المدمج لك:

```bash
# امنح صلاحية التشغيل للملف (في بيئة Linux/Mac)
chmod +x build_apk.sh

# قم بتشغيل الملف
./build_apk.sh
```

سيقوم الملف فوراً ببناء تطبيق الويب باستعمال `npm run build` الحصري ونقل الناتج لمجموعات أصول الهاتف الذكي.

### 2️⃣ فتح المشروع في Android Studio
1. قم بفتح برنامج **Android Studio**.
2. اختر **Open an Existing Project** (فتح مشروع موجود).
3. حدد المجلد الفرعي المسمى **`android`** الموجود في جذر هذا المشروع واضغط OK.
4. انتظر بضع ثوانٍ حتى يقوم Android Studio بمزامنة ملفات Gradle وتحميل المكونات اللازمة. لن تواجه أي أخطاء بإذن الله حيث تم تجميع وضبط المكتبات لتتوافق بدقة (Material 3 و Kotlin 1.9.22).

### 3️⃣ تشغيل التطبيق على الهاتف
* قم بتوصيل هاتفك الأندرويد بالكمبيوتر عبر سلك USB مع تفعيل وضع تصحيح الأخطاء (USB Debugging).
* اضغط على زر **Run** (الأيقونة الخضراء للتشغيل ◀️) في شريط الأدوات العلوي لـ Android Studio.
* سيتم تثبيت التطبيق وتفعيله فوراً على شاشة هاتفك مع شاشة ترحيبية رائعة!

### 4️⃣ استخراج ملف الـ APK النهائي (Release) جاهزاً للنشـر
لإنشاء نسخة APK كاملة وجاهزة لتوزيعها وتثبيتها بشكل مستقل أو رفعها لمتجر Google Play:

1. في Android Studio، اختر من القائمة العلوية: **Build** ثم **Build Bundle(s) / APK(s)** ثم اضغط **Build APK(s)**.
2. سيقوم النظام بتجميع التطبيق مع الضغط والتحسينات البرمجية Proguard.
3. عند الانتهاء، ستظهر لك رسالة منبثقة تفيدك بنجاح البناء، انقر على كلمة **Locate** لتجد ملف الـ **`app-release-unsigned.apk`** أو **`app-debug.apk`** الجاهز للتثبيت على أي جهاز مباشرة!
